# Employee-Data-Analysis
### Project Overview
This dashboard provides a comprehensive analysis of workforce composition, highlighting employee demographics, departmental distribution, diversity metrics, and work location trends. The goal is to leverage this data for strategic decision-making in hiring, inclusion, and workforce planning.

### Key Insights:
#### 1. Employee Status Breakdown:
- Total Employees: 22,211
- Active Employees: 90.72% (20,150)
- Terminated Employees: 9.28% (2,061)
#### 2. Gender Distribution:
- Male: 10,264 active
- Female: 9,332 active
- Non-Conforming: 554 active
#### 3. Work Location Trends:
- Headquarters Employees: 68.14% (15,134)
- Remote Employees: 22.58% (5,016)
- Terminated Headquarters Employees: 7.12% (1,581)
- Terminated Remote Employees: 2.16% (480)
#### 4. Departmental Representation:
- Largest departments: Engineering (6.7K), Marketing (3.3K), HR (1.8K)
- Smallest departments: Legal (0.3K), Analytics (0.1K)
#### 5. Diversity Metrics:
- White: 100% of workforce
- Two or More Races: 3.65K
- Black or African American: 3.62K
- Asian: 3.56K
- Hispanic or Latino: 2.50K
